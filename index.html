About the project:

This clone of Instagram was made as an assignment of a course PROG8185 "Web Technologies" of a Mobile Solutions Development program at Conestoga College, Kitchener, ON, Canada.

Project allows to create an account, add posts with images and like them just like the real Instagram does! All users' and posts' data is stored in a MySQL database. Search is NOT working.

Sample images for testing the functionality: elephants.zip.
